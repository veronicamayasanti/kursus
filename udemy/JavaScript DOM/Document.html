Document

Documen merupakan representasi object dari halaman web

Dalam satu halaman web terdapat satu document dan kita tidak perlu membuatnya secara manual karena otomatis akan ada di browser

untuk mengakses Document kita bisa gunakan object document

document object

document secara otomatis dibuat dalam browser ketika membuka halaman web, sehingga kita tidak perlu membuat object document secara manual

kita bisa langsung menggunakan kata kunci document

Document Property

Document memiliki banyak property, property dalam document bisa digunakan untuk melihat semua data yang terdapat dalam document halaman web

Document Method

Banyak method yang terdapat di document digunakan untuk memanipulasi data DOM, misalnya membuat Node, Element, Attribute atau mengambil dan menyeleksi Node di dalam document
